# ueb.config.js

```
module.exports={
 	"projectType":"web",
	"outputName":"dist",
	"htmlName":"index",
	"scssName":"index",
	"cssMin":false,
};
```
	
* **projectType**

项目类型，有web和pc两种

* **outputName**

执行`ueb pro`打包的文件夹名

* **htmlName**

html文件名

* **scssName**

scss文件名

* **cssMin**

生产环境下css是否压缩
